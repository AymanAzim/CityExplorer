# CityExplorer

CityExplorer is an interactive application that allows users to explore cities, discover points of interest,
and plan personalized itineraries.

The project is made with the typescript language  and the framework react-native. 
CityExplorer uses also free external APIs to provide dynamic city data, including maps, points of interest, restaurants, and events,
making the exploration experience interactive and update.

---

# Foursquare API Key Setup

This project uses the Foursquare Places API to search places and retrieve photos.

For security reasons, the API key is NOT included in this repository.


## 1. Get your Foursquare API Key

Create a free account and generate your API key here:
https://developer.foursquare.com/


## 2. Insert your API key in the file FourSquares.ts

 const FOURSQUARE_API_KEY = "YOUR_API_KEY"

Example:

 const FOURSQUARE_API_KEY = "ABC123..."


## Security note

The API key is intentionally excluded from the repository to prevent misuse.
Never commit your personal API keys to GitHub.
